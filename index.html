In this experiment, you will solve mini Sudoku puzzles.

The rules of Sudoku are to fill in all the digits so that no duplicates appear in any row, column or bold-bordered square.

This version of the interface uses keyboardevents and mouseevents.

To insert a number use the number keys and use arrow keys to move through the cells

Shortcuts are visible to the user when they hover mouse over the buttons on the bottom.

Shortcuts used are 'a' for clear,'s' for check,'d' for previous, 'f' for next and use Backspace key to erase a number in the cell.

Practice on puzzle #1 and #2. Then starting with puzzle #3, solve as many puzzles as quickly and correctly as you can.
